# Кнопка «Отправить сообщение» в Google Таблице

Сообщение для клиента (колонка G) можно отправить в Telegram из таблицы.

## Шаг 1. Обновить таблицу (добавить колонки H и I)

Если таблица создана давно и в ней нет колонок **«Отправить»** и **«Телефон»**, один раз выполните в папке проекта:

```bash
python update_sheet_headers.py
```

Скрипт запишет в первую строку заголовки A1:I1, включая H «Отправить» и I «Телефон». Новые строки из монитора уже содержат эти колонки; старые строки просто получат заголовки, ячейки H и I останутся пустыми — их можно заполнять вручную.

## Шаг 2. Флажки «Отправить» в каждой строке (Apps Script)

1. Откройте вашу Google Таблицу.
2. Меню **Расширения** → **Apps Script**.
3. Удалите весь код в редакторе и вставьте содержимое файла **`docs/SheetSendButton.gs`**.
4. Сохраните (Ctrl+S), закройте вкладку Apps Script.
5. Обновите страницу таблицы (F5). В меню появится **«Номера»**.
6. Один раз выберите **Номера** → **Добавить флажки «Отправить» во все строки**. В колонке H (Отправить) во всех строках с данными появятся флажки — по одному на каждую строку.
7. При первом запуске скрипта разрешите доступ: «Проверить разрешения» → выберите аккаунт Google → «Разрешить».

**Как пользоваться:** отметьте флажок в колонке H в тех строках, где нужно отправить сообщение. Затем на компьютере выполните:

```bash
python send_sheet_messages.py
```

Скрипт отправит сообщения в Telegram и запишет в H результат («Отправлено» или ошибку). Отмеченный флажок (TRUE) скрипт воспринимает как «отправить».

**Дополнительно:** в меню **Номера** есть пункты «Отправить по этой строке» и «Отправить по выбранным строкам» — они ставят «1» в H без флажков. Рисованная кнопка «Отправить» (Вставка → Рисунок) может вызывать скрипт `markRowToSend`: тогда сначала выделите ячейку в нужной строке, потом нажмите кнопку.

---

## Как это устроено

- **Колонка H «Отправить»** — значение `1` или `да` означает: отправить сообщение по этой строке.
- **Колонка I «Телефон»** — если в D «Отправитель» пусто, укажите номер (+79...) — отправка пойдёт по номеру (поиск в Telegram).
- Сообщение берётся из колонки G «Сообщение для клиента» (с подставленным номером).
- Отправка идёт через вашу пользовательскую сессию Telegram (тот же вход, что и у монитора): в личку по @username (Отправитель) или по номеру (Телефон).
